# Hersemita

Cross-country coaching dashboard for tracking runner activities and communicating with parents.

## Features

- **Coach Dashboard**: Visual analytics, performance trends, activity management
- **Runner Portal**: 6-digit code access for uploading runs (GPX/FIT/TCX)
- **Parent Messaging**: Mass SMS notifications (Twilio integration)
- **Data Visualization**: Charts and performance tables for tracking improvement

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase
- Clerk Authentication
- Recharts
- Twilio (SMS)

npm install
